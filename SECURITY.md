# Security Policy

## Reporting a Vulnerability

If you discover a security issue, please report it privately before opening a
public issue.

- Email: `contact@mccollum.dev`
- Subject: `igw-cli security report`

Please include:

- A clear description of the issue and impact.
- Steps to reproduce.
- Affected version/commit.
- Any suggested mitigation.

## Handling

- Reports are acknowledged as quickly as possible.
- A fix and release plan will be communicated after triage.
- Please avoid public disclosure until a fix is available.
